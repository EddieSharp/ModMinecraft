INSTRUCCIONES PARA PREPARAR EL ENTORNO DE TRABAJO:

1. Descargar el código de forge.
2. Descomprimirlo.
3. Mover la carpeta resultante al workspace.
4. Abrir un terminal y movernos hasta tal carpeta.
5. Ejecutar “./gradlew eclipse” o “gradlew.bat eclipse” para windows.
6. Ejecutar “./gradlew setupDecompWorkspace” o “gradlew.bat setupDecompWorkspace” para windows.
7. Copiar src.
